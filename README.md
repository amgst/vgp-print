# Print Services Website

This is a code bundle for Print Services Website. The original project is available at https://www.figma.com/design/ZnSKYyRtIwOwr5Lf12x2nz/Print-Services-Website.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deploying to Vercel

1. Push your code to a Git repository
2. Log in to your Vercel account
3. Create a new project and import your repository
4. Vercel will automatically detect the Vite project and configure the build settings
5. Click "Deploy" and your site will be live!

The build command is `npm run build` and the output directory is `dist`.
